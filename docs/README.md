# Antigravity Chess Engine

A Python-based chess engine with a React UI.

## Features
- Minimax Search with Alpha-Beta Pruning
- Iterative Deepening
- Move Ordering (Basic)
- Transposition Table (Placeholder)
- UCI Protocol Support
- React + Tailwind UI

## Setup

### Engine
1. Install requirements:
   ```bash
   pip install -r requirements.txt
   ```
2. Run CLI:
   ```bash
   python ui/cli/play.py
   ```
3. Run Server (for UI):
   ```bash
   python ui/server.py
   ```

### UI
1. Navigate to `ui/react-app`:
   ```bash
   cd ui/react-app
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```
